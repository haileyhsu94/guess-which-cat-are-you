# C-3PAW — Which Cat Are You? 🐱

A retro-futuristic personality quiz that diagnoses which cat you are, based on
100% real internet cat folklore and 0% actual science.

Built as a single self-contained `index.html` — no frameworks, no build step.
Live cat photos courtesy of [Cataas](https://cataas.com).

Try to turn it off. Go ahead. The cat won't let you.
